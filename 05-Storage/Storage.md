# Azure Storage

## Overview
Azure Storage offers highly available and durable data storage solutions.

## Topics Covered
- Storage Accounts
- Blob Storage
- File Storage
- Queue and Table Storage
- Redundancy options

## Key Concepts
- Blob storage is used for unstructured data
- Storage accounts must have unique names
- Data is encrypted by default

- Configure lifecycle rules
