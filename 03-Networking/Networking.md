# Azure Networking

## Overview
Azure Networking enables secure communication between cloud resources.

## Topics Covered
- Virtual Networks (VNet)
- Subnets
- Network Security Groups (NSG)
- Public and Private IPs
- Azure DNS
- VNet Peering

## Key Concepts
- VNets provide isolation
- NSGs control traffic
- Subnets divide networks logically
