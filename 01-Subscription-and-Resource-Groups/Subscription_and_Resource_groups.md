# Subscription and Resource Groups

## Overview
Azure Subscriptions and Resource Groups are the foundation of Azure resource management and billing.

## Topics Covered
- Azure Subscription overview
- Subscription types (Free, Pay-As-You-Go, Enterprise)
- Resource Groups
- Resource lifecycle management
- Naming conventions and tagging
- Cost management basics

## Key Concepts
- Subscription is a billing and access boundary
- Resource Groups act as logical containers
- All resources in a group share the same lifecycle

- Create a resource group using Azure CLI
